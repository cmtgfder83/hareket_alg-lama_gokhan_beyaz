# MPU6050 Sensörü ile Hareket Algılama Sistemi

Bu proje, **STM32L031K6** mikrodenetleyicisi ve **MPU6050** ivmeölçer sensörü kullanılarak geliştirilmiştir.

## Teknik Özellikler
* **Mikrodenetleyici:** STM32L031K6
* **Sensör:** MPU6050 (I2C Haberleşme)
* **Haberleşme:** UART (115200 Baud) üzerinden seri terminale veri aktarımı

## Bağlantı Şeması
* **SCL:** PA9 (D1) pinine bağlı
* **SDA:** PA10 (D0) pinine bağlı
* **VCC:** 3.3V
* **GND:** Ground
